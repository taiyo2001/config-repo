### tpm (Tmux Plugin Manager)
tpmを導入
```
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

tpm管理のプラグインをインストール

tmux内で以下を実行
```
pfx + I
```
参考：[document](https://github.com/tmux-plugins/tpm)
